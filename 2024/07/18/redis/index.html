Redis

Redis 是一个开源（BSD许可）的，内存中的数据结构存储系统，它可以用作数据库、缓存和消息中间件。它支持多种类型的数据结构，如 字符串（strings）， 散列（hashes）， 列表（lists）， 集合（sets），带有范围查询的 有序集合（sorted sets） ， bitmaps， hyperloglogs 和 地理空间（geospatial） 索引。 Redis 内置了 主从复制（replication）， LUA脚本（Lua scripting）， LRU驱动事件（LRU eviction）， 事务（transactions） 和不同级别的 磁盘持久化（persistence）， 并通过 Redis哨兵（Sentinel） 和自动 分区（Cluster）提供高可用性（high availability）

Redis 是 key-value 型数据库，键（Key）命令是 Redis 中经常使用的命令。常用的键命令如下所示：

命令 描述
DEL 用于删除 key
DUMP 序列化给定 key ，并返回被序列化的值
EXISTS 检查给定 key 是否存在
EXPIRE 为给定 key 设置过期时间
EXPIREAT 用于为 key 设置过期时间，接受的时间参数是 UNIX 时间戳
PEXPIRE 设置 key 的过期时间，以毫秒计
PEXPIREAT 设置 key 过期时间的时间戳(unix timestamp)，以毫秒计
KEYS 查找所有符合给定模式的 key
MOVE 将当前数据库的 key 移动到给定的数据库中
PERSIST 移除 key 的过期时间，key 将持久保持
PTTL 以毫秒为单位返回 key 的剩余的过期时间
TTL 以秒为单位，返回给定 key 的剩余生存时间(
RANDOMKEY 从当前数据库中随机返回一个 key
RENAME 修改 key 的名称
RENAMENX 仅当 newkey 不存在时，将 key 改名为 newkey
TYPE 返回 key 所储存的值的类型

Redis五大基本数据类型

  1. String（字符串）

    添加，查询，追加，获取长度，判断是否存在的操作

    自增，自减操作

    截取，替换操作

    设置过期时间，不存在设置操作

    mset，mget操作

    总结：String 是redis中最常用的一种数据类型，也是Redis中最简单最基础的数据类型，表面上他是字符串，但是其实他也可以灵活的标识字符串，整数，浮点数3种值。 Redis会自动识别这3种值

  2. Hash

    hset（添加hash）， hget(查询)，hgetall(查询所有)，hdel(删除hash中指定的值)，hlen（获取hash的长度），hexists(判断key是否存在)

    hkeys (查询所有的key) ，hvals(查询所有的值)，hincrby(指定hash的filed自增)，hsetnx(存在不添加)

    总结： 比String更适合存对象

  3. Set(集合) 元素唯一

    sadd(添加)，smembers(查看所有的元素)，sismember(判断元素是否存在)，scard(查看长度)，srem(移除元素)

    srandmember(抽随机)

    spop（随机删），smove（指定移动）

    sdiff(差集)，sinter(交集)，sunion(并集)

    总结： 可以实现共同好友，共同关注等功能需求

  4. ZSET（有序集合）

    zadd(添加)，zrange(查询)，zrangebyscore(排序 小-大)，zrevrange(排序 大-小)，withscores(是否查询权重)

    zrem(移除元素) ，zcard(查看元素的数量)， zcount（指定区间内的元素数量）

    总结： 需要做排序的数据，可以使用zset， 成绩表，工资表，朋友圈动态 （Timeline）
